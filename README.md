# language-profile
A Wycliffe language-profile connecting our audience with information, stories, and stats for these
languages through an immersive experience that incorporates giving and email newsletter sign-
up CTAs.

## Technologies Used
* HTML
* CSS
* Javascript
* JQuery
* MySQL
* Node.js, Express
